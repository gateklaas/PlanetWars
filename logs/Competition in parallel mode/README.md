* Adaptive bot
* Beam bot
* Bully bot
* Greedy bot
* Lookahead bot
* Minimax bot
* Random bot

### Result ###

A score can be computed by subtracting the number of losses from the number of wins ($score = wins - losses$).
This score gives a general idea of how well a bot performed.
With this score, the bots can be ranked.
